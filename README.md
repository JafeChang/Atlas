# Atlas
Atlas 是一个以 **个人信息聚合与结构化** 为核心目标的技术项目，旨在通过自动化数据采集、分类、抽取与索引，将分散在不同公开来源的信息持续汇聚为可检索、可分析、可演进的个人信息库。  项目整体偏向 **Python 技术栈**，强调可长期运行的任务调度（如 cron）、可迁移的本地与云端部署，以及 **LLM 与轻量模型协同** 的处理架构：  * 使用大语言模型进行低频但高价值的语义理解与结构化抽取 * 使用小模型或传统算法完成高频任务（去重、排序、重排、过滤等） * 支持逐步扩展数据源、模型能力与存储规模  Atlas 的目标不是一次性爬取，而是构建一个**可持续生长的信息聚合系统**，为个人研究、长期知识积累和跨领域信息跟踪提供稳定底座。
