## 📋 文档分类说明

### 📂 只增文档 (Accumulative)
- **特征**：内容只添加，不修改历史
- **范围**：技术架构、合规准则、知识库
- **更新**：只能追加内容，保留历史

### 📂 可变更文档 (Mutable)
- **特征**：需要定期更新
- **范围**：使用文档、项目状态、任务管理
- **更新**：版本控制，记录变更

### 📂 开发日志 (Development Logs)
- **特征**：按时间序列记录
- **范围**：变更记录、测试结果、研究笔记
- **更新**：时间顺序，持续追加
